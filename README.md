# 🧠 Memory Match Game

A classic card matching game built with HTML, CSS, and JavaScript. Challenge your memory and see how fast you can find all the matching pairs!

## 🚀 Features
* **Interactive Gameplay:** Click to flip cards and find matches.
* **Responsive Design:** A clean, grid-based layout that looks great on different screens.
* **Game Logic:** Automatically shuffles cards and handles matching/mismatching logic.

## 🛠️ Technologies Used
* **HTML5:** Structure of the game board.
* **CSS3:** Grid layout and card flip animations.
* **JavaScript:** Game mechanics, shuffling logic, and state management.

## 🎮 How to Play
1. Open the game in your browser.
2. Click on any two cards to flip them.
3. If they match, they stay visible.
4. If they don't match, they flip back after a second.
5. Keep going until all cards are matched!

## ⚙️ How to Run Locally
1. Clone this repository.
2. Open the `index.html` file in your preferred web browser.
3. (Recommended) Use the "Live Server" extension in VS Code for the best experience.
